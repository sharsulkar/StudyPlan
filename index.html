Semester 1: Fabric, Python, KQL

14-week plan (~10–12 hours/week) focusing on Microsoft Fabric fundamentals, lakehouse engineering with PySpark, Python for data work, and KQL for Real-Time Analytics. This semester lays the foundation for DP-600 and DP-700.

Week Main Theme Fabric Tasks Coding / KQL Tasks Optional AI Tasks Est. Hours

- Work through high-level Fabric overview modules 		- Refresh Python basics (syntax, control flow, functions)
- Set up Jupyter / notebooks environment 		- Skim a short ML overview (types of ML, basic terminology) 10–12

- Create sample lakehouse, warehouse, and semantic model 		- Start pandas: DataFrames, reading CSVs, simple aggregations - Buffer week (no AI required) 10–12

Master-Style Study Program – Course Catalog

Semester 1 Semesters 2–6 Courses

This catalog lists all courses in the two-year plan across Microsoft Fabric, coding, Azure Solutions Architect, and Azure AI tracks. Each course includes goals, key skills, and suggested resources so you can align them with certifications (DP-600, DP-700, AZ-104, AZ-305, AI-102).

Fabric Track

FAB-101 – Fabric Fundamentals & OneLake Concepts (3 cr)

Level: Intro · Focus: Fabric architecture, workloads, medallion concepts
Goals
  • Build a mental model of Microsoft Fabric, key workloads, and medallion architecture.
  • Navigate workspaces and OneLake confidently.
Key skills
  • Fabric architecture: capacities, workspaces, items, OneLake; link to Power BI and Synapse.
  • Workload overview: Data Engineering, Data Factory, Data Warehouse, Real-Time Analytics, Data Science, Power BI.
  • Medallion concepts (bronze/silver/gold) and when to choose lakehouse vs warehouse.
Resources
  • Microsoft Learn Fabric overview and DP-600 learning paths.
  • DP-600T00-A course syllabus as a topic map.

FAB-120 – Lakehouse, Spark, and Delta in Fabric (3 cr)

Level: Core · Focus: PySpark, Delta, lakehouse patterns
Goals
  • Build lakehouses using PySpark and Delta aligned with DP-700 expectations.
Key skills
  • Lakehouse creation and table design with partitions.
  • PySpark DataFrames, joins, aggregations, window functions.
  • Delta tables and basic structured streaming.

FAB-140 – Dataflows Gen2 & Pipelines (2 cr)

Level: Core · Focus: SaaS-style ingestion and orchestration
Goals
  • Use Dataflows Gen2 and Pipelines to ingest and orchestrate data in Fabric.

FAB-160 – Warehouse & Semantic Modeling (3 cr)

Level: Core · Focus: star schemas, warehouse, semantic models, DAX
Goals
  • Design a warehouse and semantic model that supports analytics workloads efficiently.

FAB-220 – Governance, Security & DevOps in Fabric (3 cr)

Level: Advanced · Focus: security, governance, CI/CD
Goals
  • Operate Fabric as a governed, production platform with multiple environments.

FAB-240 – Healthcare Data Engineering on Fabric (4 cr)

Level: Advanced Project · Focus: HL7/FHIR, PHI, healthcare analytics
Goals
  • Apply Fabric to healthcare data feeds and PHI-aware pipelines.

FAB-390 – DP-600 & DP-700 Exam Workshop (2 cr)

Level: Exam · Focus: exam prep and gap-closing
Goals
  • Close gaps and get exam-ready for Fabric Analytics Engineer and Fabric Data Engineer certifications.
